# StupidDB

StupidDB is a highly customizable lightweight in-memory key-value database written in 100% Kotlin.

Pros:
1. Highly customizable
2. Lightweight
3. High performance
4. Multiplatform
5. Scalable
6. Easy to use
7. Direct access
8. Open source
9. Support any kind of data structure

Cons:
1. Stupid
